## 🛠️ Setup Instructions

Follow these instructions to set up and run this project locally on your machine.

### Prerequisites

Ensure you have the following installed on your system:
*   **Node.js** (v18.0.0 or higher recommended)
*   **npm** (bundled with Node.js) or another preferred package manager (Yarn, pnpm, or Bun)

### 1. Clone the Repository
Clone the repository using Git:
```bash
git clone https://github.com/CopyCoderJay/banking_page.git
```

### 2. Navigate to the Directory
Switch your terminal's current working directory to the cloned repository folder:
```bash
cd banking_page
```

### 3. Install Dependencies
Run the installation command to fetch all required npm packages:
```bash
npm install
# or yarn install / pnpm install / bun install
```

### 4. Start the Development Server
Launch the local Next.js development server:
```bash
npm run dev
# or yarn dev / pnpm dev / bun dev
```

Open your browser and navigate to **[http://localhost:3000](http://localhost:3000)** to view the live local version of the app.

## 🚀 Production Deployment

### Building Locally
To test the production build locally:
```bash
npm run build
npm run start
```

### Deploying to Vercel
1. Push your changes to your own GitHub fork or repository.
2. Sign in to your [Vercel Dashboard](https://vercel.com).
3. Import the repository and click **Deploy**. Next.js project settings will be detected automatically.
